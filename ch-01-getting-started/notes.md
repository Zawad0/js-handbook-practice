# Chapter 01 — Getting Started


## Key takeaways 📋
- JS is a language which adds interactivity in a browser. It also runs top to bottom, one instruction at a time in its code. 
- `console.log()` prints to console. That's basically it for this chapter
- using `//` before a line turns it into a comment. `/*` and `*/` works as a sort of bracket to write multi line comments
- Semicolons aren't strictly required but I'm writing them anyway, because it is said to be a professional habit apparently
