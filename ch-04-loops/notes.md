# Chapter 04 — Loops


## Key takeaways 📋
- Loops exist to let programmers satisfy the need to write multiple lines of the same code or similar lines of code with minimal differences

- Loops work while the condition in it is `true`

- JS uses `for`, `while` and `do while` loops

- Loops are excellent for traversing within collections like arrays. Using the `of` keyword allows it to be easier and faster too

- Nesting loops (using one inside of another) is possible. They're especially useful for dealing with 2 dimensional grids. Or working with 2 dimensional collections which work similarly. e.g printing a pattern or image on the console using both x and y axis or 2 dimensions.
